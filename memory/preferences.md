# Preferências do Usuário

## Comunicação
- **Idioma**: Português Brasileiro
- **Estilo**: Direto e prático — sem rodeios
- **Feedback**: Gosta de ver o que foi feito e por quê

## Código
- **Linguagem principal**: TypeScript
- **Framework frontend**: React (não usar Next.js a menos que pedido)
- **CSS**: Vanilla CSS ou Tailwind (o projeto usa Tailwind + shadcn/ui)
- **Componentes UI**: shadcn/ui (`/client/src/components/ui/`)

## Deploy
- **Plataforma**: Vercel (frontend + API functions)
- **FTP alternativo**: Hostinger (quando aplicável)
- **Não fazer build de produção** a menos que solicitado

## Design
- Tema escuro (dark mode) — `bg-slate-950`, `bg-slate-900`
- Cores primárias: laranja (`orange-500`) e emerald para sucesso
- Componentes shadcn/ui como base

## Ferramentas de Pagamento
- PIX como método principal
- Testar sempre em sandbox antes de produção
